## Dev Utils

Dev utils to be shared across Quachtli projects and packages.

## Usage

Install the module:

```bash
$ yarn add @quachtli/dev-utils
```

And import it in your project:

```js
const quachtliDevUtils = require("@quachtli/dev-utils");
```

### Install Modules

```bash
$ yarn install
```

### Build

```bash
$ yarn build
```

### Lint

```bash
$ yarn lint
```

### Clean

```bash
$ yarn clean
```
