## Dev Utils

Dev utils to be shared across NoteStream projects and packages.

## Usage

Install the module:

```bash
$ yarn add @notestream/dev-utils
```

And import it in your project:

```js
const devUtils = require("@notestream/dev-utils");
```

### Install Modules

```bash
$ yarn install
```

### Build

```bash
$ yarn build
```

### Lint

```bash
$ yarn lint
```

### Clean

```bash
$ yarn clean
```
